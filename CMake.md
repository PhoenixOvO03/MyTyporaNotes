# CMake

## 环境准备

检查环境

```shell
cmake --version
```



## 创建CMakeLists.txt文件

```shell
touch CMakeList.txt
```



## 常用命令

+ 注释
  + `# echo` 注释行
  + `# [[echo]]` 注释块
+ `cmake_minimum_required(版本号)` 指定CMake最低版本
+ `project(项目名)` 指定项目名称
+ `add_executable(可执行项目名 源文件名称)` 指定 工程生成可执行程序
